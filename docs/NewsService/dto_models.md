        "id": 1, // идентификатор новости на сайте АГПУ

# Модель подробной новости
```
{
        "id": 1, // идентификатор новости на сайте АГПУ
        "title": "Заголовок новости",
        "description": "Полный текст новости",
        "date": "dd.MM.yyyy",
        "images": [
            // массив строк содержащий URL до изображений, принадлежащей новости
        ]
    }
```
